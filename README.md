# 🦭 Seal Helper

**Веб-платформа для підтримки і захисту тюленів**

![Seal Helper Preview](https://ichef.bbci.co.uk/news/1536/cpsprodpb/1415/live/2eabedf0-58c0-11f0-bcb7-a356aa14e90d.jpg.webp)

## 📋 Опис проєкту

**Seal Helper** — це веб-сайт, створений для підтримки та захисту тюленів через:
- 💰 Систему онлайн-донатів
- 📰 Поширення інформації про морських ссавців
- 🎁 Програму винагород для донаторів
- 📧 Підписку на новини та оновлення
- 👥 Залучення волонтерів до захисту тюленів

## 🎯 Мета проєкту

Створити централізовану онлайн-платформу для:
1. Збору пожертв на захист тюленів
2. Організації волонтерської допомоги
3. Інформування громадськості про проблеми морських ссавців
4. Прозорого звітування про використання донатів

## ✨ Основні функції (MVP)

### 1. Система донатів
- Вибір із готових варіантів сум ($10, $25, $50, $100)
- Можливість вказати власну суму
- Система винагород за донати:
  - **$10** — Цифрова листівка з тюленем 📧
  - **$25** — Сертифікат опікуна тюленя 📜
  - **$50** — Футболка Seal Helper 👕
  - **$100** — Плюшевий тюлень + всі попередні подарунки 🧸
- Email-підтвердження протягом 1 хвилини після донату

### 2. Розділ новин
- Актуальні історії про врятованих тюленів
- Інформація про реабілітаційні центри
- Анонси волонтерських програм
- Система дат публікацій

### 3. Інформаційні розділи
- Статистика проєкту (врятовані тюлені, зібрані кошти, волонтери)
- Опис місії та цінностей
- Контактна інформація

## 🚀 Інструкція з запуску

### Локальний запуск

1. **Клонуйте репозиторій:**
```bash
git clone https://github.com/ilmnayccmm/seal_helper_fullstack
cd seal_helper_fullstack
```

2. **Відкрийте проєкт:**
   - Використайте локальний сервер:

```bash
# За допомогою Python
python -m http.server 8000

# За допомогою Node.js (http-server)
npx http-server
```

3. **Перейдіть у браузері:**
```
http://127.0.0.1:5173
```

### Деплой на GitHub Pages

1. Завантажте код на GitHub
2. Перейдіть у Settings → Pages
3. Оберіть гілку `main` та папку `root`
4. Збережіть налаштування
5. Ваш сайт буде доступний за адресою: `https://your-username.github.io/seal-helper/`
